1. What this project does

Purpose

Downloads podcast episodes from Spotify URLs (no auth required — via PodcastIndex RSS feed lookup), transcribes locally with faster-whisper, and writes Obsidian-compatible markdown files. Daily automation (Windows Task Scheduler) runs sync + post-processing: generates a 20-point LLM summary per episode and updates an HTML mindmap with new cards, auto-curated into the correct theme section.
